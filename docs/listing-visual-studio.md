# Listing 视觉工作台：图文整图直出

## 产品目标

视觉套图以一张白底产品图为产品真值，自动使用第 ① 步的采集与产品洞察、第 ② 步的 Listing
文案，并接受用户手动创意需求。图片模型一次生成产品、场景、构图、辅助设计、英文文字和排版，
不再采用“无字底图 + 本地叠字”作为新工作台的生产路径。

目标交付物是一组风格和色系统一、单张任务清楚、产品外观可信、文字准确可读且具有完整销售叙事的
Amazon 商品套图或 A+ 模块。

## 输入优先级

1. **产品真值**：上传白底图优先，其次使用通过白底检测的采集主图。它是产品外形、比例、颜色、
   材质、Logo、接口、配件和数量的唯一视觉依据。
2. **事实层**：采集的标题、要点、产品信息、AI 洞察和已生成 Listing 文案。
3. **手动创意需求**：对受众、风格、场景、色系、重点和禁用项的要求，优先于自动风格建议，但不能
   覆盖产品真值、事实合规和 Amazon 主图规则。

## 生成流程

1. **建立产品视觉身份**：分析品类、刚柔属性、尺度、材质反应、自然交互方式和不可变细节。
2. **建立整套视觉 DNA**：确定五色色板、白平衡、光线、材质、字体气质、装饰语言、画面密度和禁用项。
3. **设计销售叙事**：每张图回答一个购买问题，分别承担白底主图、核心利益、真实使用、细节、规格、
   对比、包装或信任任务；不同图片不得重复构图。
4. **编译最终提示词**：每张提示词包含产品身份锁、镜头与物理关系、共享视觉 DNA、精确公开文案、
   字体层级与位置、8% 安全边距和负向约束。
5. **图文一次直出**：商品副图和 A+ 的文字由图片模型写入最终像素；白底主图保持纯白无营销文字。
6. **质量闸门**：逐张检查技术质量、产品一致性、文字逐字准确、可读性与商业完成度；整套再检查色系、
   设计语言和叙事一致性。

## 提示词原则

- `FINAL ARTWORK COPY` 中的每个字符串必须只出现一次，逐字符一致，不得翻译、改写、缩写、遗漏或重复。
- 标题优先控制在 2–5 个英文单词；辅助说明不超过 9 个英文单词。短文案能显著提高图片模型的拼写成功率。
- 图上文字只允许来自已确认文案和产品事实；证据说明、审核意见和内部提示不得进入画面。
- 五色色板用于背景、承托面、道具、环境调色和文字，不得用于重绘产品本身；白底主图始终使用 `#FFFFFF`。
- 构图应随产品物理属性变化：软品需要真实折叠和垂坠，空间类产品需要可信占地和人体尺度，硬件设备需要
  保留按键、接口、开孔和精确几何。
- 禁止默认使用霓虹光轨、悬浮玻璃、科幻平台、假 UI、虚构证明、漂浮产品和泛化的“电影级奢华”装饰。

## 返工与版本

- OCR 或视觉质检发现错字、漏字、重复字、额外文字或产品失真时，系统携带具体错误自动重画一次。
- 修改图上文案、文字区、版式或最终提示词会使当前成图失效；旧图进入历史版本，用户需要重新生成该张。
- 单张历史版本最多保留 8 个。整套导出保留人工复核清单；未复核图片以 `DRAFT` 标记。

## 商用闸门

以下条件全部满足后才可标记为可交付：

- 产品一致性达到硬门槛，且人工核对外观、接口、Logo、配件和数量。
- 目标公开文案逐字符准确，无额外营销文字，手机缩略图下仍可读。
- 所有数字、认证、对比与效果有可追溯事实依据。
- 每张图片通过单图质检，整套通过色系、设计系统和叙事一致性质检。
- 主图与对应站点画布符合 Amazon 规则。
